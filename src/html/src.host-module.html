loadKernel(system, RP, operators)

source code 

Loads the kernel to the GPU and readies it to be run.

The kernel code from kernel.src is imported and read. The binary program that will be sent to the GPU is built and memory allocations on the GPU are done.

Parameters:
  • system (physical_systems class) - The physical system that is considered.
  • RP (run_params.RunParams) - The run parameters that have been set for the run.
  • operators (tuple of strings) - The operators that will be calculated in the run.
Returns: pimc_kernel.PIMCKernel
Class containing the stuff needed by the function runKernel.

runKernel(pimcKernel)

source code 

Runs the kernel with the system and the run parameters

This will start the kernel on the GPU. When the GPU is done desired data is fetched

Parameters:
  • pimcKernel (pimc_kernel.PIMCKernel) - The class that is returned from loadKernel.
Returns: RunKernelResults
