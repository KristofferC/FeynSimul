Class RunKernelResults

source code

RunKernelResults is a class where results from runKernel are dumped.

After a kernel run the following variables are available:

Always:

  • runTime (float) - The time in seconds it took to execute the kernel.
  • acceptanceRate (float)- - Average of how often path changes where accepted or rejected in the Metropolis step.

If getOperator in run_params.RunParams is enabled:

  • operatorMean (float) - The average value of the operators calculated by the threads.
  • operatorStandardError (float) - The standard error in the operatorMean.

If binsEnabled and returnBinCounts in run_params.RunParams are enabled:

  • binCountNormed (numpy array) - Calculated probability density in the interval specified in the run parameters.

If returnPaths in run_params is enabled:

  • paths (numpy array) - All the current active paths for the threads.
