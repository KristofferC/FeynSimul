Class RunParams

source code

Stores variables that are needed to perform a PIMC run.

Variables that always need to be defined in the class before a run are:

  • N (int) - The number of nodes in a path.
  • beta (float) - Euclidian time difference between first and last node in the path.
  • nbrOfWalkers (int) - The number of paths that should be worked on simultaneously.
  • operatorRuns (int) - The number of times the operator should be calculated before returning the results from the GPU.
  • metroStepsPerOperatorRun (int) - The paths between operator measurements should be uncorrelated so it makes no sense to calculate determines how many changes should be made to the path before the operator should be calculated along the whole path.

There are also options to make regarding what type of changes that should be made to the path. They are listed below together with their corresponding parameters.

  • enableBisection (bool) - Uses the bisection algorithm to make changes to the path.
    • S (int) - The total bisection depth'
    • enableParallelizePath (bool) - This makes it so that many threads work on the same path simultaneously. The number of threads that will work on a path is N / (2**S).
  • enablePathShift (bool) - Takes an interval of the path and offsets the whole interval.
    • PSAlpha (float) - The offset of the interval of the path will be a random number between -PSAlpha and PSAlpha.
  • enableSingleNodeMove (bool) - Offsets a single node in the path.
    • alpha (float) - The offset of the node of the path will be a random number between - alpha and alpha.

Caclulation of the probability density is enabled with binsEnabled (bool). Variables that affect how the probability density is calculated are

  • xmin (float) & xmax (float) - A hypercube with the sides xmin to xmax where the probability density will be calculated inside.
  • binResolutionPerDOF - Determines how many discretization points to do in a degree of freedom. The total number of bins will be binResolutionPerDOF to the power of the degrees of freedom in the system.

You can chose what type of memory the paths that the threads are working on should be stored with enableGlobalPath (bool). With enableGlobalPath they are stored in the global memory and else in the local memory.

If returnPaths (bool) is enabled the paths are fetched from the GPU after a run.
